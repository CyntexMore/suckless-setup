# suckless-setup

My suckless setup.

# Installation

To install __dwm__, __st__, __dmenu__ and __slstatus__ you just have to do the following:
```
git clone https://github.com/CyntexMore/suckless-setup/

cd dwm
cd st
cd dmenu
cd slstatus

sudo make clean install
```
